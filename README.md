# Library Management System (SE Assignment)

## Overview

This project is a simple Library Management System developed using

Agile methodology and Test-Driven Development (TDD).

## Features

- Add books to the library

- Borrow and return books

- Generate library status report

## Technologies Used

- Python 3

- unittest framework

- Git (branching, tagging)

## Agile Sprints

- Sprint 1: Book Registration

- Sprint 2: Borrow & Return Books

- Sprint 3: Library Report

## How to Run Tests

python -m unittest -v
